# 📚 Simple LMS System

A Learning Management System (LMS) built as my first university project.  
It includes role-based dashboards for Admin, Manager, Teacher, and Student with full CRUD functionality for academic management.

---

## 🚀 Features

- 🔐 Role-based authentication (Admin, Manager, Teacher, Student)
- 🧑‍🎓 Student management system
- 👨‍🏫 Teacher management and course assignment
- 📚 Course and department management
- 🎯 Enrollment system
- 📝 Marks management (Mid, Final, Presentation)
- 📊 Attendance tracking system
- 📈 Separate dashboards for each user role

## 🛠️ Tech Stack

- Python (Flask)
- PostgreSQL
- HTML, CSS, JavaScript
- Basic Pandas (for data handling)

---

## 🏗️ System Architecture

- Flask handles backend routing and logic
- PostgreSQL used for database management
- HTML/CSS/JavaScript for frontend
- Role-based access control for security
- Basic MVC-style structure

---

## 🗄️ Database Tables

- users
- students
- teachers
- courses
- departments
- enrollments
- attendance
- marks
- teacher_courses
